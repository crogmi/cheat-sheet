HTML/CSS Cheat Sheet

Topic Definition Example
HTML
<h1> to <h6> Represents header elements and can be used for titles or subtitles throughout the web page <h1>The Title of This Web Page</h1>
<img> Inserts an image element into HTML with the source and the alternative text (in case the image does not load). Image elements do not require a closing tag <img src="[URL]" alt="[alternative text]">
<a> Links a reference outside or within the web page <a href="[URL]">[Text used to serve as the link]></a>
<button> Serves as a button that operates its defined function, such as a submit button <button type="submit">Submit</button>
<input> Can be used for radio and checkbox buttons <label for "[text]"> <input id="[text]" type="radio" value="[text]" name="[text consistent across all inputs]">[Text] </label>
<div> Serves as a general purpose container and divider. Should not replace semantic elements. <div class="[class name]></div>
CSS
* Represents the universal selector to apply CSS to all elements in the web page * { font-family: Verdana; }
class Classifies an element within the specified class. Prepend class name with a . within CSS. .element-class { font-size: 16px; }
id Classifies an element within the specified id. An element's id can only have a single value and only be used once per page. Prepend an id name with a # within CSS. #element-id { font-weight: bold; }
Pseudo-class selectors Factors such as the interaction, site navigation, and position in the DOM can all give elements a different state with pseudo-class. :fact, :visited, :disabled, :active, and :hover
!important Can be applied to specific declarations. It will override any style no matter how specific it is. It should almost never be used as it is hard to overwrite. #element-id { font-weight: bold !important; }
